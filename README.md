# Cricket Performance Analytics Dashboard

Interactive Power BI dashboard analysing India vs Australia ODI batting statistics using ESPN Cricinfo Statsguru data.

## Features

- Power BI KPI dashboard
- Dynamic player filtering
- Strike rate analysis
- Batting average analysis
- Match performance metrics
- Interactive slicers and visuals
- Power Query web data ingestion

## Data Source

Data sourced directly from ESPN Cricinfo Statsguru using Power BI Web Connector:

https://stats.espncricinfo.com/ci/engine/stats/index.html?class=2;home_or_away=1;opposition=2;team=6;template=results;type=batting

## Technologies Used

- Power BI
- Power Query
- DAX
- Web Data Connector
- Sports Analytics
